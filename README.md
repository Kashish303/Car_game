# 🚗 AI Self-Driving Car Simulation

## 📌 Overview

This project simulates a self-driving car using deep learning.
The model learns from gameplay data and predicts steering direction.

---

## ⚙️ Features

* Data collection using simulation
* CNN-based model training
* Real-time AI driving
* Obstacle avoidance system

---

## 🧠 Workflow

1. Run `collect_data.py` → generate training data
2. Run `train_model.py` → train model
3. Run `run_ai_car.py` → test AI driving
